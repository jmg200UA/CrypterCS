Hemos desarrollado una aplicación para encriptar y desencriptar con AES128, haciendo uso de los frameworks 
vue.js (que nos ha permitido desarrollar la interfaz gráfica de nuestro proyecto),
electron.js (a partir del cual hemos podido desarrollar la aplicación gráfica de escritorio),
una base de datos propia que hemos creado a partir de un fichero en plano JSON creado por nosotros,
y la librería de algoritmos criptográficos estándar CryptoJS, a partir de la cual hemos realizado el cifrado y descifrado de archivos.
 
En la memoria que hemos realizado incluimos más detalles sobre las herramientas y referencias a estas,
que nos han ayudado a documentarnos y a aprender a usarlas.

Hemos creado una aplicación de escritorio para que corra en Windows, por lo tanto solo es necesario
hacer doble click sobre el archivo .exe y ejecutamos la aplicación.

La aplicación consta de la ventana de encriptación y la de desencriptación.
